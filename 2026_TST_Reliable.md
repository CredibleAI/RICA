# Reliable Multi-Agent Reasoning with Verifiable Rewards

**Mentor**: Dr. [Tomasz Steifer](https://scholar.google.pl/citations?user=Sjw4GpgAAAAJ&hl=en&oi=ao)

Multi-agent LLM systems are becoming increasingly popular approach to automatic problem solving. However, we lack good understanding of how to combine multiple agents' capabilities beyond simple increase in compute. There is also some mixed evidence. For instance,  a recent NeurIPS paper reported that a simple majority vote beats several AI debate protocols across multiple tasks. In any case, both simple debate and majority vote can fail when agents converge on shared but incorrect reasoning paths and become mutually reinforcing.

This project will develop and evaluate mechanisms for coordinating, selecting, and controlling LLM agents so that multi-agent reasoning systems become more reliable than simple debate or majority voting. We will use techniques from game theory, social choice, control theory, information theory, and formal epistemology. We will use automated theorem proving as a primary experimental environment.
